# LAN 的游戏素材工具箱

这是一个给独立游戏开发者用的小工具箱。它不需要服务器，不需要安装软件，也不会上传你的图片或视频。

所有处理都在你的浏览器本地完成。

## 直接打开

下载这个仓库后，双击打开：

[打开工具箱 index.html](./index.html)

如果你是在 GitHub 网页上看这个 README，点上面的链接可能会进入文件预览，而不是直接运行工具。最稳的方式是：

1. 点击 GitHub 的 `Code`。
2. 选择 `Download ZIP`。
3. 解压后双击 `index.html`。

如果仓库开启了 GitHub Pages，也可以把 `index.html` 作为在线页面直接打开。

## 里面有什么工具

### 抠图工具

适合把图片背景去掉，导出透明 PNG。

支持：

- 拖拽或选择图片
- 黑底去除
- 纯色背景去除
- 边缘连通背景去除
- `Shift + 点击` 多次取样背景色
- 同步缩放和中键拖动画布
- 导出透明 PNG

常见用法：

1. 导入图片。
2. 选择一种抠图模式。
3. 如果是纯色背景或边缘连通背景，点击原图取样背景色。
4. 背景有多个颜色时，按住 `Shift` 再点击追加取样色。
5. 调整阈值、羽化、去边等参数。
6. 导出 PNG。

### 视频转精灵图

适合把一段 `mp4` 或 `webm` 动画转成游戏里常用的精灵图。

支持：

- 导入视频
- 均匀抽取指定帧数
- 生成单帧 PNG 序列
- 生成 spritesheet PNG
- 生成 JSON 元数据
- 下载单帧 PNG zip

默认参数：

- 抽取帧数：24
- 每行列数：6
- 行数自动计算
- fps：24，只写入 JSON

### 像素风转换器 / Pixel Style Converter

适合把 AI 生成图、普通图片或伪像素图，转换成更接近游戏可用的像素风素材。

支持：

- Classic Pixel / 经典像素
- Modern Pixel / 现代独游像素
- HD Pixel / 高清像素
- VFX Pixel / 特效像素
- Advanced Clean 算法
- Pixelit Style 算法
- 自定义调色板
- 透明 PNG alpha 保留
- 导出实际低分辨率 PNG
- 导出最近邻放大的预览 PNG

常见用法：

1. 导入图片。
2. 选择处理预设。
3. 调整像素宽度和高度。
4. 需要更强像素感时，试试 `Classic Pixel`。
5. 做技能图标、敌人、普通素材时，试试 `Modern Pixel`。
6. 做发光特效时，试试 `VFX Pixel`。
7. 导出实际 PNG 或放大预览 PNG。

## 自定义调色板格式

像素风转换器支持输入调色板，例如：

```text
#000000,#ffffff,#2b6cff,#ffcc33
```

如果有效颜色少于 2 个，工具会自动回退到自动取色。

## 隐私说明

这个工具不会上传文件。

图片和视频只会在你的浏览器里处理。刷新页面或关闭浏览器后，导入的素材不会保存在工具里。

## 适合谁

这个工具适合：

- 不会写脚本、但想快速处理素材的独游开发者
- 想把 AI 图简单整理成游戏素材的人
- 需要临时抠图、抽帧、做像素风预览的人

它不是 Photoshop，也不是专业动画软件。它的目标是快速、简单、本地可用。
